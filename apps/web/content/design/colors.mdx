## Usage

```scss
.element {
  background: var(--color-gray-bg);
  color: var(--color-gray-text);
  transition: 150ms ease;
  transition-property: background, color;

  &:hover {
    background: var(--color-gray-bg-hover);
    color: var(--color-gray-text-contrast);
  }
}
```
